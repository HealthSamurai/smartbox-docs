# Getting Started with Smartbox

Smartbox is a SMART on FHIR application launcher and testing platform that helps you develop, test, and deploy SMART on FHIR applications.

## What is Smartbox?

Smartbox provides:

- **SMART App Launcher** - Launch and test SMART on FHIR applications
- **OAuth 2.0 Server** - Built-in authorization server for SMART workflows
- **App Gallery** - Manage and organize your SMART applications
- **Testing Tools** - Validate SMART app behavior and compliance

## Quick Start

1. Contact us at [health-samurai.io/contacts](https://health-samurai.io/contacts) to get access
2. Configure your SMART application
3. Test your application using the launcher
